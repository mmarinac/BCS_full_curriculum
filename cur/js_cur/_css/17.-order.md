A final tip for this introduction to flexbox is that it is possible to not just reverse the order of flex items, but also to specify an order with the order property.

This property takes an integer as the value and allows you to define a display order for the flex items. It is important to remember that this doesnâ€™t change the logical order and any text to speech browser will read out the order as in the source. If you actually should re-order the source do that rather than just moving things around with flexbox:

	.card:nth-child(1) {
	  order: 3;
	}

	.card:nth-child(2) {
	  order: 1;
	}

	.card:nth-child(3) {
	  order: 2;
	}
